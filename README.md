## Run Locally

Clone the project

```bash
  git clone https://github.com/DevStack06/react-whatsapp-web
```

Go to the project directory

```bash
  cd react-whatsapp-web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
